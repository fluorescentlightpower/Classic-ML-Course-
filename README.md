Ноутбук "Анализ данных ЛВ" содержит EDA исходного набора данных и позволяет экспортировать обработанные наборы для задач регрессии и классификации в бинарные файлы .joblib. Эти файлы предназначены для загрузки в другие ноутбуки с соответствующими задачами. Также в файлы .joblib экспортируются модели для регрессии CC50 и IC50, потому что они требуются для части заданий по регрессии и классификации SI.
